# Python Project with DDD architecture

This repository is an example to implement DDD architecture on a Python Rest API.

## Tech Stack 
- Fastapi
- redis
- Poetry 
- Docker

## Folder Structure

## Execute the project

1. Clone this repo
2. Open a terminal
3. In the terminal execute `make dev`
4. Access to the api in `localhost:5000/docs`
